feat: Add Makefile targets for rustc test flakes and configs

This commit introduces new targets to the main Makefile to automate the creation and management of Nix flakes and bootstrap configurations for different rustc versions.

New targets include:
- `generate-rustc-test-flakes`: Creates directories and `flake.nix` files for specified rustc versions.
- `generate-rustc-test-configs`: Generates `generated_config.toml` files for each rustc version using `bootstrap-config-generator`.
- `clean-rustc-test-flakes`: Removes the generated flake directories and config files.

This streamlines the process of testing different rustc toolchains within the Nix environment.
